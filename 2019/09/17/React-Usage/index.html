    • Refs相关

    Refs主要内容为，可以在父组件精确调用子组件中的具体DOM元素/函数.(相当于是跨层级，也可以用第三方库例如，jQuery，d3等来达到)

    React官方示例，Refs and the DOM

    还可以类似于props的方式，继续讲Refs向子组件继续转发，相当于父组件可以调用子…子组件的DOM元素.

    React官方示例,Refs转发

    Render

    当props相同的时候，react仍然会进入render部分(函数式就会进入整个函数),但不会比较render内容的DOM元素是否相同，会比较渲染中所用到的state、props等是否相同从而提高效率.

    也就是说，大多数情况下，props相同时会进入render但不会重新渲染.
